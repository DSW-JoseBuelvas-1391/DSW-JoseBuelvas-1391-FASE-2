Recursos educativos con Etiquetas Basicas de HTML, atributos y valores

Este sitio a sido diseñado para suplir la necesidad de la presencialidad en la clases de los estudiantes. Esta dirigido a todas las
instituciones educativas del pais, esto como resultado a la necesidad de elaborar recursos educativos digitales que nos permitan dinamizar
las metodologias de aprendizaje autonomo y dirigido. Teniendo en cuenta el avance de la tecnologia podemos enfocar la enseñanza y aprendizaje
de maneras virtuales. Estos proyectos, es diseñado con contenidos multimediales e interactivos, basados en HTML5, CSS3 entre otras tecnologías
para que el aprendizaje sea practico y dinamico.

HTML5

Es un lenguaje de programacion que utiliza una serie de codigos llamados etiquetas
que van definiendo los elementos que componen una pagina web: texto, imágenes, etc.
Esas etiquetas seran interpretadas por un programa navegador de internet. Pero no solo
se limita a ser un lenguaje de etiquetas HTML que solo permite definir elementos basicos,
sino que combina nuevas etiquetas de lenguaje HTML, propiedades CSS3, Javascript y algunas
otras tecnologias.

Etiquetas basicas

  • doctype: Primera linea del codigo presente en cualquier documento HTML, indica al navegador
    las especificaciones HTML
  • header: Define la cabecera de la pagina
  • footer: define el pie de pagina
  • nav: define un elemento de navegacion de la pagina como el menu principal o secundarios
  • section: encierra el codigo de una seccion del documento
  • aside: delimita el contenido secundario
  • article: permite indicar el contenido de la pagina web
