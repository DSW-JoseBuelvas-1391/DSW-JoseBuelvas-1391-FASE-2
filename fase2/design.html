Proposito del sitio web

El propósito de este sitio web es con el objetivo de crear contenido para conceptos educativo de HTML.
En el cual se profundizará en el concepto del tema Flex Box dando a conocer la funcionalidad que tiene
en el diseño de las páginas web.

Borrador del contenido

Flex Box

Es un metodo de diseño de pagina unidimensional que sirve para organizar elementos en filas o columnas.
Los elementos que contenido crecen para llenar el espacio adicional y se reducen para adaptarse al espacio mas pequeño.
con la realizacion de esta pagina se revelaran todas las caracteristicas basicas y su funcionalidad.

El diseño CSS Flexbox es un módulo de CSS que define un modelo de caja, optimizado para el diseño de interfaces de usuario.
Propiedades en CSS

  • Flex-basis
  • Flex-dorection
  • Flex-flow
  • Flex-grow
  • Flex-shrink
  • aside: delimita el contenido secundario
  • article: permite indicar el contenido de la pagina web
